# Configurtion Files

## ZSH

- Install using the software manager

'''bash
sudo apt install zsh
'''

- Change the shell:

'''bash
chsh -s $(which zsh)
'''

- Install complements for zsh:

'''bash
sudo apt install zsh-syntax-highlighting zsh-autosuggestions
'''

## NVIM

- Use the update_nvim to install the newest version.
